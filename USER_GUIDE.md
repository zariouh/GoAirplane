# Airplane Mode Scheduler App - User Guide

## Introduction
Welcome to the Airplane Mode Scheduler app! This guide will help you understand how to use the app to automatically schedule airplane mode during your sleep hours and other times when you want to disconnect.

## Getting Started

### Installation
1. Download the APK file from the release folder
2. Enable "Install from Unknown Sources" in your device settings
3. Open the APK file to install the application
4. Grant the necessary permissions when prompted

### Initial Setup
When you first open the app, you'll need to:
1. Grant the WRITE_SETTINGS permission (required for airplane mode control)
2. Grant notification permissions
3. Set up your default sleep schedule

## Main Features

### Home Screen
- **Current Profile**: Shows your active profile and its status
- **Quick Actions**: Buttons to immediately enable or disable airplane mode
- **Skip Next**: Skip the next scheduled airplane mode change
- **Edit Schedule**: Quickly navigate to edit your schedule
- **Upcoming Schedules**: View all upcoming scheduled airplane mode changes

### Schedule Management
- **Sleep Schedule**: Set your regular sleep hours
  - Enable Time: When airplane mode should turn on
  - Disable Time: When airplane mode should turn off
  - Repeat Options: Daily, Weekdays, Weekends, or Custom
- **Add New Schedule**: Create additional schedules for different scenarios

### Profile Management
- **Sleep Profile**: Default profile for sleep hours
- **Create New Profiles**: Set up profiles for different scenarios (meetings, travel, etc.)
- **Profile Settings**:
  - Notification time before changes
  - Notification after changes
  - Associate with specific schedules

### Settings
- **Theme**: Switch between light and dark themes
- **Notifications**: Configure notification preferences
- **Permissions**: Request or review app permissions

## Tips for Best Use
1. **Battery Optimization**: Make sure the app is excluded from battery optimization to ensure reliable scheduling
2. **Regular Updates**: Check for app updates to get the latest features and improvements
3. **Multiple Profiles**: Create different profiles for weekdays and weekends if your sleep schedule varies
4. **Test First**: Test the app during the day before relying on it for your actual sleep schedule

## Troubleshooting
- **Airplane Mode Not Toggling**: Check that the WRITE_SETTINGS permission is granted
- **Missing Notifications**: Verify notification permissions are granted
- **Schedules Not Persisting**: Ensure the app is not being force-closed by battery optimization

## Privacy Information
- This app does not collect or transmit any personal data
- All schedules and settings are stored locally on your device
- No internet connection is required for the app to function

We hope you enjoy using the Airplane Mode Scheduler app to improve your digital wellbeing and sleep quality!
